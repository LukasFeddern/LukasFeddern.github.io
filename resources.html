Additional Resources

R

General introductions/Free online courses

R-Core Team: An Introduction to R

The guide An Introduction to R encompasses a lot of the material covered in this tutorial. The focus is on applications in base R, including data types, reading in data, basic statistics, and programming, as well as packages, and navigating OS facilities. This is a free resource directly from the R-Core Team, which maintains and distributes R.

Hands-on Programming with R

The book Hands-on Programming with R provides a great introduction to R and programming in R. The link takes you to the free online online version of the book.

Applied Epi

The Interactive R Tutorial is an interactive resource for learning R. It requires registration but is completely free and easily accessible. The course is self-paced and includes data preparation, cleaning, and the basics of data visualization in R.

UW Course: CSSS 508

The University of Washington offers a free online course covering the basics of R CSSS 508. The course page offers futher resources and great videos.

Tidyverse

The Tidyverse is a fairly new environment in R. Its structure and logic differ quite substantially from base R and other packages in R. You might find it more intituitive than other packages.

R for Data Science

This is the free online version of the book R for Data Science by Hadley Wickham and Garett Grolemund. It provides a holistic introduction to R in general and the tidyverse specifically. It includes topics such as data import, visualization, management, and modeling. It also provides an introduction to R Markdown, a way to produce pdf, html, or word documents using R Studio.

Data visualization

R Graphics Cookbook

This is the free online version of the popular R Graphics Cookbook. The book offers a basic introduction into R, and graphing in R. It focusses on ggplot(), the graphing function in tidyverse. It covers bar graphs, line graphs, scatter plots, summarized data distributions, annotations and output presentation among other things.

Social network analysis

This is a brief introduction to social network analysis in R. We briefly touch on importing, cleaning, visualizing and analysis of network data.

Introduction to social network analysis with R

This training session by Pablo Barbera is an excellent short introduction into network visualization in R.

User’s Guide to Network Analysis in R

This free e-book by Douglas Luke provides an extensive overview of social network analysis in R, including code examples and theory.

Statistical Analysis of Network Data with R

This free e-book explains data structures, statistical methods, and visualization of network data. It includes code examples in R.

Troubleshooting

R is not easy and sometimes some of the error messages you can get are cryptic. That is where the R community comes in. If you need help with errors, unintended output, or instructions, Stackoverflow, the R Studio Community Forum, Quick-R by Data Camp, Quick-R by Data Camp, R Bloggers, and ProgrammingR, are good resources to get support. Sometimes a quick google search might also do the trick.

Stata

General introductions/Free online courses

Fundamentals of data analysis and visualization

This repository was developed by Aaron Chafetz, et al. and provides a great introduction to Stata. It covers data wrangling, data analysis, and data visualization in Stata.

Troubleshooting

If you have issues and or need instructions for Stata, the official Stata website provides help. Furthermore, Statalist is a helpful community resource for Stata users.
